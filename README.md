# AML Open Framework — Pitch Deck

Live deck: **https://tomqwu.github.io/aml_open_framework_demo/**

19-slide HTML presentation for the [AML Open Framework](https://github.com/tomqwu/aml_open_framework) project.

## What's here

This is a static mirror of `docs/pitch/deck/` from the main repo, deployed via GitHub Pages so anyone with the link can flip through the deck without cloning.

## Usage

- Open `index.html` (or the live URL above)
- Arrow keys / Space / 1-9 / Home / End to navigate
- `P` to print to PDF
- Hash routing — e.g. `#11` jumps to the Audit & Evidence slide

## Source

Slide source HTML lives in [`tomqwu/aml_open_framework`](https://github.com/tomqwu/aml_open_framework) under `docs/pitch/deck/`. This repo is a deployment target only — for changes, edit the main repo and re-sync.
